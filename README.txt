Text look up program that focuses on quick access, look up and copy to clipboard

Created for lookup and usage of common responses at my current position.
Features
	- Stateless client
	- Multiple user access
	- Live search
	- Auto copy to clipboard

Uses SQLite for a database. System.Data.SQLite ADO.NET adaptor required to be installed on development machine.

http://system.data.sqlite.org/index.html/doc/trunk/www/index.wiki

System.Data.SQLite.dll exports to local on compile so client only needs exe and dll to run
